TmdaCode

Collection of TMDA code, or links to 3rd party packages related to TMDA.

TMDA-related software

  • TmdaCgi - a beta-release program for managing your TMDA account over the web.

  • Bill Duncan has written a SquirrelMail plugin for TMDA.

  • Mate Wierdl has written tmda4ezmlm, a script that sets up TMDA for an ezmlm mailing list.

  • Brian Koontz has written tmda-ezplm, scripts to automate the management of the TMDA pending queue via e-mail.

  • ToddJacobs has written install_tmda.sh, a script to automate TMDA installation into a home directory. The script features support for the mailx, pine, and mutt MUA, and generates well-documented examples using the user's own information at runtime.

  • David Morison has written TMDA Tools, another SquirrelMail plugin for version 1.4.x, offering features similar to TmdaCgi.

  • ToddJacobs has written qbrowse.tk, This new GUI interface for managing the pending queue has been tested on Debian GNU/Linux and Macintosh OS X, and currently requires Tcl/Tk 8.4 and with Tix support.
