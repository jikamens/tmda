Upgrading TMDA

Each TMDA release comes with an UPGRADE file that lists incompatibilities with prior releases. You need to make sure you satisfy these before upgrading your TMDA.

Do this by reading UPGRADE from the bottom up, and start at the first applicable section. Then follow all the instructions until you reach the top. Not everything will apply to your particular configuration in which case you can skip them.
