TMDA Configuration

  • ConfigurationOverview - gets you started on how to configure TMDA. You should read this first and use it as your guide. Some of the pages below are linked to from this document.

  • The TMDA distribution contains a set of example ~/.tmda/* files to get you started in the contrib/dot-tmda/ directory. Here is the README from that directory.

Other Configuration Pages

  • PreConfiguration - configuration prerequisite components such as your SMTP server.

  • ServerConfiguration - how to filter your incoming mail with TMDA.

  • ClientConfiguration - the client side of TMDA that integrates with your mail reader.

  • ConfigurationVariables - a list of all the TMDA configuration variables.

  • FilterSpecification - documents the format and syntax of TMDA's filter files.

  • FilterSources - further explanation of the filter files.

  • TmdaHowtos might be worth reading as well.
