Getting Support for TMDA

  • Don't forget to read TmdaDocumentation

  • TroubleShooting - this is a good place to start if you are having problems or have questions.

  • TmdaFaq - frequently asked questions about TMDA.

  • MailingLists - public TMDA discussion lists and searchable archives.

  • TmdaCommercialSupport - you can hire private consultants to help you with or develop TMDA for you.

  • There is a #tmda channel on irc.freenode.net where some of the developers can often be found.
